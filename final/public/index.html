Quizgen

Host onsite quizzes online.

Linked directly to Google Sheets. Easier to make and share.

Get started.

Create a Google Sheet

Duplicate this sample sheet. It won't work if the format isn't exactly the same, so be careful.
Click on File, Publish to the Web.

Publish to the Web

Select the name of the sheet, and click on Publish.

Get the URL

Copy the Google Sheet URL from the browser input and paste it into the field up top.
Press the Quiz button.

...good to go.

The grid should appear, complete with all the content from the linked sheet. Access the quiz anywhere using
http://quizgenfinal.firebaseapp.com/?sheet=sheeturl.
